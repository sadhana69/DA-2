# DA-2
Mini Banking System

A simple C++ console-based banking system that allows users to create accounts, deposit and withdraw money, check balance, and view recent transactions.

Features
Create account
Deposit & withdraw money
Balance enquiry
View last 5 transactions
Reports (total money, low balance)
File storage using accounts.txt and transactions.txt
Technologies Used

C++, File Handling, OOP Concepts

How to Run
Compile the code
Run the executable
Use the menu to perform operations
File Structure
Main code file
accounts.txt
transactions.txt
